#### if

> True 或 False

- ==  值是否相等
- ！= 不相等
- 多条件判断
  - and  
  - or
- in 检查特定值是否包含在列表中
- not in 不包含

```python
if xxx:
	op1
[elif yyy:]
	op2
[else:]
	op3
```

#### while(循环)

```python
while condition:
	op1
```

- break 退出所有循环
- continue 跳过本次循环
